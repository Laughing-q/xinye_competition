```shell
cd xinye_competition
```

## Prepare data
```shell
python tools/prepare_data.py --regressor-data --detector-data --generate-pair
```

## Train regressor
```shell
python tools/train_regressor.py
```

## Train detector
```shell
python tools/train_detector.py
```

## Inference
```shell
python inference.py
```
